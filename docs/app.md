# AI生成ネイティブなVJアプリ

## 概要
AI技術を用いた軽量で扱いやすいVJアプリ
- GLSL(Shadertoy),p5.js,Three.jsなどの小さいコード・シーンをAIによって自由自在に生成
- 動画なども用いれる。
- 動画的なアセットもAIによって生成可能（PythonサーバにI2VやT2Vを要求できる、あるいは普通にダウンロード）
- 動画や、シーンを組み合わせて、簡単にVJができる。
    - 例えば、動画を背景にして、クロマキーで抜くなどして、GLSLで生成されたシーンを重ねることも可能。
    - シーン、動画を切り替えたり、動画の再生位置を変えたりすることも簡単にできる。
- オリジナルのESP32 LEDに対して、動画やシーンを投影することも可能。
- 生成されたコードは、ユーザーが自由に編集可能。
- 映像を出力するウィンドウと、制御するウィンドウは別にする。
- LED への出力は、カメラを用いて、LEDの位置を認識して、そこに映像を投影する（映像からLEDの位置のピクセルを抽出、それをLEDに送信）。（別添Pythonスクリプトと同じ機能を持って、それに映像を映せる）

## 技術スタック
- フロント　: React, TypeScript, Vite , Threejs p5.js glsl(Shadertoy)
- Rust + Tauri
